Laboratorium di Jurusan Teknik Informatika Polindra
Lab Jaringan Komputer
Pada laboratorium ini diselenggarakan praktikum Sistem Operasi, Jaringan Komputer 1, admin dan keamanan jaringan, jaringan nirkabel dan Jaringan Komputer 2.

Fasilitas 1 Server & 30 PC Workstation, RouterBoard Mikrotik, Cisco Switch Managable 3065, Cisco Router 2901, Juniper Router, Juniper Firewall, RF, Antena grid, Omni, dan Yagi, wirelles router, Wimax.
Sistem Operasi Windows Server 2008, Ubuntu Server, Redhat, Windows 7 dan Linux Ubuntu
Aplikasi Microsoft Office, Oracle VM VirtualBox, Mikrotik OS, Packet Tracer, dan Wireshark
Aktifitas - Praktikum Sistem Operasi
- Praktikum Jaringan Komputer 1
- Praktikum Jaringan Komputer 2
- Praktikum Komunikasi Data
- Pekerjaan tugas mata kuliah, untuk kegiatan penelitian dan pengabdian pada masyarakat

Lab Pemrograman
Di laboratorium ini diselenggarakan praktikum Pemrograman Terstruktur, Algoritma, software engineering, pemrograman berbasis obyek. Workshop ini juga berfungsi sebagai common support bagi kebutuhan penggunaan komputer. Mahasiswa dapat menggunakan komputer yang ada untuk mengerjakan tugas dan akses internet sesuai aturan yang telah ditetapkan


Fasilitas 1 Server & 30 PC Workstation
Sistem Operasi Windows Server 2008 dan WIndows 7
Aplikasi Microsoft Office, Visio, Visual Studio.Net, Hummingbird Exceed (X windows emulator), My SQL, SQL Server dan Oracle DBMS, Java, eclipse, netbean, devC++, Rational Roose, microsoft project.

- Praktikum pemrograman berbasis obyek
